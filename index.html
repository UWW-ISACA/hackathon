What Is a Hackathon?

A Hackathon is a collaborative event where individuals gather to work on software projects or a virtual capture-the-flag event within a limited timeframe. Participants form teams, brainstorm ideas, and/or develop their projects from scratch. At the end of the hackathon, teams present their projects, competing for prizes, and gain recognition.

How Can I Get Involved?

For our participants, we've set up a linktree that has the event rules, link to the event discord, team signup page, and project submission page. We ask that each of the participants review each of these links to best prepare for the event!
